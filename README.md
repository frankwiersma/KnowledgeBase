### Active Directory Security

#### Common Vulnerabilities

*   Certificate Services (ADCS) misconfigurations.
*   Delegation issues and misconfigured permissions.
*   SCCM attack vectors.
*   Kerberos vulnerabilities (e.g., Kerberoasting, relaying).
*   Password policy weaknesses.
*   Group Policy misconfigurations. [https://twitter.com/JimSycurity/status/1737464410367578304](https://twitter.com/JimSycurity/status/1737464410367578304)
*   Domain trust issues.
*   Enabled Guest account.
*   Insecure delegations (using ADeleg).



#### Attack Mitigation

*   Implement Local Administrator Password Solution (LAPS).  [https://twitter.com/NathanMcNulty/status/1790838141290729652](https://twitter.com/NathanMcNulty/status/1790838141290729652)
*   Use Windows Hello for Business. [https://twitter.com/NathanMcNulty/status/1790838141290729652](https://twitter.com/NathanMcNulty/status/1790838141290729652)
*   Configure Windows Firewall defaults.
*   Restrict admin tool access with AppLocker.
*   Monitor domain controller access.
*   Implement strong password policies.
*   Deploy security baselines.
*   Patch Apex One (Trend Micro) vulnerabilities.
*   Monitor domain controller access.
*   Restrict adding computer accounts to admins.  [https://twitter.com/EricaZelic/status/1783388609266999525](https://twitter.com/EricaZelic/status/1783388609266999525)
*   Monitor default and custom privileged AD groups.
